1. Project Overview
The goal of this project is to build a modern, responsive, and secure website for RCN Mission Hospital, inspired by the structure, UI/UX, and user flow of MissionPartnersForChrist.org.
The website will promote the hospital’s medical missions, outreach programs, sponsorships, and care initiatives (Arome Care, Dinna Care, Doc Care), while allowing admins to manage content, gallery, blog, and reports through a secure backend dashboard.

6. Blog Module
URL Structure:
- Blog List: /blog
- Blog Post: /blog/slug-name

Database Table: blog_posts
Fields: id, title, slug, excerpt, content, image, author, created_at

.htaccess Rules:
RewriteEngine On
RewriteRule ^blog/([^/]+)/?$ blog/post.php?slug=$1 [L,QSA]
RewriteRule ^blog/?$ blog/index.php [L,QSA]
RewriteRule ^([^/]+)/?$ $1.php [L,QSA]

7. Admin Dashboard

---

8. Security Architecture
- HTTPS enforced via .htaccess
- X-Frame-Options, X-Content-Type-Options headers
- Session regeneration on login
- CSRF protection with tokens
- PDO prepared statements
- Password hashing (bcrypt)
- Rate-limited login attempts
- reCAPTCHA v3 optional
- IP blacklisting on failed logins
- Daily backups and audit logs

---
